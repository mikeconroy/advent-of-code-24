# Advent of Code 2024

Day 1-8 solutions in Rust.
Day 9+ solutions in Golang.

To run all solutions:
`./run.sh`

To run Rust solutions:
```
cd rust/
cargo run --release
```

To test Rust solutions:
```
cd rust/
cargo test
```

To run Go solutions:
```
cd go/
go run ./...
```

To test Go solutions:
```
cd go/
go test ./...
```
